================================================================================
1. TỔNG QUAN DỰ ÁN
================================================================================

HistoVision là một ứng dụng Flutter đa nền tảng (Android, iOS, Web, Desktop) 
giúp học sinh học lịch sử Việt Nam thông qua việc tạo video AI từ văn bản hoặc 
ảnh. Ứng dụng sử dụng công nghệ AI tiên tiến để biến các sự kiện lịch sử thành 
video sống động.

TÍNH NĂNG CHÍNH:
- Tạo video 5 giây hoặc 15 giây từ văn bản mô tả sự kiện lịch sử
- Tạo video từ ảnh tư liệu lịch sử
- Tạo 3 video đồng nhất song song (tính năng nâng cao)
- Xem video với chế độ stereo 3D
- Lưu video vào gallery
- Hỏi đáp về lịch sử với AI (Gemini, OpenAI)

4. QUY TRÌNH TẠO VIDEO TỪ VĂN BẢN (5 GIÂY VÀ 15 GIÂY)
================================================================================

QUY TRÌNH TẠO VIDEO 5 GIÂY:

Bước 1: Người dùng nhập văn bản
   └─> Ví dụ: "Trận Bạch Đằng năm 938"
   
Bước 2: Xử lý văn bản (home_screen.dart)
   └─> _normalizeDescription(): Chuẩn hóa văn bản
   └─> Kiểm tra văn bản không rỗng
   
Bước 3: Tạo prompt với GPT-5 (openai_service.dart)
   └─> generatePrompt(text): Gửi văn bản đến GPT-5
   └─> GPT-5 phân tích và tạo prompt chi tiết với:
       - 2-3 scenes (mỗi scene 1-2 giây)
       - 5 yếu tố mỗi scene: Environment, Camera, Character/Action, 
         Soundtrack/SFX, Visual Style
       - Thoại tiếng Việt trong dấu ngoặc kép
       - Tổng độ dài: 5 giây
   
Bước 4: Tạo video với Sora 2.0 (ai_video_service.dart)
   └─> generateVideoFromText(prompt): Gửi prompt đến AIVideoAuto API
   └─> API trả về videoId
   └─> Polling trạng thái video mỗi 10 giây
   └─> Khi video hoàn thành, trả về downloadUrl
   
Bước 5: Lưu và hiển thị video
   └─> Tự động lưu vào gallery (gallery_service.dart)
   └─> Chuyển đến màn hình phát video (video_player_screen.dart)

QUY TRÌNH TẠO VIDEO 15 GIÂY:

Tương tự như video 5 giây, nhưng:
- GPT-5 tạo prompt với 3-5 scenes (mỗi scene 3-5 giây)
- Tổng độ dài: 15 giây
- Gọi generateVideoFromText15s() thay vì generateVideoFromText()

```dart
Future<void> _generateVideo() async {
  // 1. Lấy và chuẩn hóa văn bản
  String text = _textController.text.trim();
  text = _normalizeDescription(text);
  
  // 2. Kiểm tra văn bản không rỗng
  if (text.isEmpty) {
    _showErrorDialog('Vui lòng nhập text để tạo video');
    return;
  }
  
  // 3. Hiển thị loading
  setState(() {
    _isGenerating = true;
  });
  
  // 4. Retry logic (tối đa 3 lần)
  const maxRetries = 3;
  int retryCount = 0;
  
  while (retryCount < maxRetries) {
    try {
      // 5. Tạo prompt với GPT-5
      final prompt = await _openAIService.generatePrompt(text);
      
      // 6. Tạo video với Sora 2.0 (5 giây)
      final videoUrl = await _videoService.generateVideoFromText(prompt);
      
      // 7. Lưu video vào gallery
      await _galleryService.saveVideo(
        videoUrl: videoUrl,
        title: text,
        sourceType: 'text',
      );
      
      // 8. Chuyển đến màn hình phát video
      Navigator.push(
        context,
        MaterialPageRoute(
          builder: (context) => VideoPlayerScreen(videoUrl: videoUrl),
        ),
      );
      
      break; // Thành công, thoát khỏi vòng lặp
    } catch (e) {
      // Xử lý lỗi và retry
      retryCount++;
      if (retryCount >= maxRetries) {
        _showErrorDialog('Lỗi khi tạo video: $e');
        break;
      }
    }
  }
  
  setState(() {
    _isGenerating = false;
  });
}
```

```dart
Future<String> generateVideoFromText(String prompt) async {
  // Thêm yêu cầu thoại tiếng Việt
  prompt += '\n\nNOTE: The dialogues in the video must be in Vietnamese.';
  
  // Chuẩn bị request body
  final requestBody = {
    'access_token': accessToken,
    'domain': domain,
    'model': 'sora_2_0',
    'privacy': 'PRIVATE',
    'prompt': prompt,
    'duration': '5',  // ← Độ dài 5 giây
    'ratio': '16:9',
    'project_id': 'default',
    'images': jsonEncode([]), // Không có ảnh
  };
  
  // Gửi request đến API
  final response = await http.post(
    Uri.parse('$baseUrl/create-video'),
    headers: {'Content-Type': 'application/x-www-form-urlencoded'},
    body: requestBody,
  );
  
  // Xử lý response
  if (response.statusCode == 200) {
    final data = jsonDecode(utf8.decode(response.bodyBytes));
    
    if (data['videoInfo'] != null) {
      final videoId = data['videoInfo']['id_base'] ?? data['videoInfo']['id'];
      
      // Polling trạng thái video
      return await _waitForVideoCompletion(videoId);
    }
  }
  
  throw Exception('Error creating video');
}
```

3. Hàm polling trạng thái video:
```dart
Future<String> _waitForVideoCompletion(String videoId) async {
  const maxAttempts = 60; // Tối đa 10 phút (60 * 10 giây)
  const pollInterval = Duration(seconds: 10);
  
  for (int attempt = 1; attempt <= maxAttempts; attempt++) {
    // Kiểm tra hủy
    if (_isCancelled) {
      throw GenerationCancelledException();
    }
    
    // Đợi 10 giây
    await Future.delayed(pollInterval);
    
    // Kiểm tra trạng thái
    final videoInfo = await checkVideoStatus(videoId);
    final status = videoInfo['status'] as String;
    
    if (status == 'MEDIA_GENERATION_STATUS_SUCCESSFUL') {
      // Video hoàn thành
      final downloadUrl = videoInfo['download_url'] as String?;
      if (downloadUrl != null && downloadUrl.isNotEmpty) {
        return downloadUrl; // Trả về URL video
      }
    } else if (status == 'PUBLIC_ERROR_UNSAFE_GENERATION') {
      // Prompt không an toàn, retry với prompt an toàn hơn
      if (onUnsafe != null) {
        return await onUnsafe();
      }
    } else if (status.contains('FAILED') || status.contains('ERROR')) {
      // Lỗi, dừng polling
      throw Exception('Video generation failed: $status');
    }
    // Nếu chưa xong, tiếp tục polling
  }
  
  // Timeout sau 10 phút
  throw Exception('Video generation timed out');
}
```

```dart
static const String _userPromptTemplate = """
Bạn là chuyên gia lịch sử Việt Nam và kỹ thuật prompt engineering cho AI 
text-to-video Sora. Nhiệm vụ của bạn là chuyển đổi chủ đề lịch sử Việt Nam 
thành prompt chi tiết cho video 5 giây, chia thành các scenes hợp lý.

### BƯỚC 2: XÂY DỰNG CÁC SCENES CHO VIDEO 5 GIÂY

Chia video thành **2-3 scenes**, phân bổ thời gian hợp lý (mỗi scene 1-2 giây):
- **Scenes đầu**: Thiết lập bối cảnh, môi trường, dẫn dắt
- **Scenes giữa**: Phát triển sự kiện, hành động chính, cao trào
- **Scene cuối**: Đỉnh điểm với thoại nhân vật, thể hiện tinh thần

### BƯỚC 3: MỖI SCENE PHẢI CÓ 5 YẾU TỐ

**1. Environment (Môi trường):**
- Thời kỳ lịch sử cụ thể (thế kỷ mấy)
- Địa điểm Việt Nam chi tiết
- Ánh sáng, thời tiết, âm thanh môi trường

**2. Camera Directions (Góc máy):**
- Loại shot: wide shot, medium shot, close-up, aerial...
- Chuyển động camera: tracking, push-in, pan, tilt...

**3. Character/Action (Nhân vật/Hành động):**
- Nhân vật: Vietnamese people with East Asian features
- Trang phục Việt Nam theo thời kỳ
- Hành động cụ thể, rõ ràng

**4. Soundtrack/SFX (Âm thanh):**
- Nhạc nền: war drums, trống chiến, sáo trúc...
- Hiệu ứng âm thanh: tiếng nước, gió, lửa...

**5. Visual Style (Phong cách hình ảnh):**
- Cinematic, photorealistic, epic...

### BƯỚC 4: THOẠI NHÂN VẬT

**BẮT BUỘC** phải có ít nhất 1 scene có thoại nhân vật:
- Thoại phải bằng **TIẾNG VIỆT** và đặt trong dấu ngoặc kép " "
- Ví dụ: "Đánh đuổi giặc Nguyên!", "Tiến lên, nhấn chìm chúng nó!"

CHỦ ĐỀ: [topic]
""";
```

2. Hàm tạo prompt:
```dart
Future<String> generatePrompt(String userText) async {
  // Thay thế [topic] trong template bằng văn bản người dùng
  final userPrompt = _userPromptTemplate.replaceAll('[topic]', userText);
  
  // Gửi request đến OpenAI API
  final response = await http.post(
    Uri.parse('https://api.openai.com/v1/chat/completions'),
    headers: {
      'Authorization': 'Bearer $_apiKey',
      'Content-Type': 'application/json; charset=utf-8',
    },
    body: json.encode({
      'model': 'gpt-5', // Sử dụng GPT-5
      'messages': [
        {'role': 'user', 'content': userPrompt},
      ],
    }),
  );
  
  // Xử lý response
  if (response.statusCode == 200) {
    final data = json.decode(utf8.decode(response.bodyBytes));
    final content = data['choices'][0]['message']['content'] as String;
    return content.trim(); // Trả về prompt đã tạo
  }
  
  throw Exception('Error generating prompt');
}
```

```dart
Future<void> saveVideo({
  required String videoUrl,
  required String title,
  required String sourceType,
}) async {
  try {
    // 1. Tải video từ URL
    final response = await http.get(Uri.parse(videoUrl));
    
    if (response.statusCode == 200) {
      // 2. Lưu video tạm thời
      final directory = await getTemporaryDirectory();
      final file = File('${directory.path}/video_${DateTime.now().millisecondsSinceEpoch}.mp4');
      await file.writeAsBytes(response.bodyBytes);
      
      // 3. Lưu vào gallery
      await Gal.putImageBytes(
        file.readAsBytesSync(),
        album: 'HistoVision',
        name: title,
      );
      
      print('✅ Video saved to gallery: $title');
    }
  } catch (e) {
    print('❌ Error saving video: $e');
    throw Exception('Failed to save video: $e');
  }
}
```

   - "Tạo Video 5 giây": Tạo video ngắn, nhanh (mặc định)
   - "Tạo Video 15 giây": Tạo video dài hơn, chi tiết hơn

4. Chờ quá trình tạo video:
   - "Đang tạo prompt từ GPT..." (5-10 giây)
   - "Đang tạo video từ prompt..." (1-5 phút)
   - Video sẽ tự động lưu vào gallery

5. Xem video:
   - Video sẽ tự động mở trong trình phát
   - Có thể xem lại từ gallery

LƯU Ý:
- Video 5 giây: Nhanh hơn, phù hợp cho preview
- Video 15 giây: Chi tiết hơn, phù hợp cho nội dung đầy đủ
- Có thể hủy quá trình tạo video bằng nút "Hủy tạo video"
- Nếu prompt không an toàn, hệ thống sẽ tự động retry với prompt an toàn hơn

================================================================================
9. XỬ LÝ LỖI VÀ RETRY LOGIC
================================================================================

CÁC TRƯỜNG HỢP LỖI:

1. Prompt không an toàn (Unsafe Generation):
   - Hệ thống tự động retry với prompt đã được làm sạch
   - Tối đa 3 lần retry

2. API quá tải (503, Overloaded):
   - Tự động retry sau 5, 10, 20, 30 giây
   - Tối đa 5 lần retry

3. Timeout:
   - Video không hoàn thành sau 10 phút
   - Hiển thị thông báo và cho phép thử lại

4. Lỗi mạng:
   - Kiểm tra kết nối Internet
   - Thử lại sau khi kết nối ổn định

10. TỐI ƯU HÓA VÀ BEST PRACTICES
================================================================================

1. XỬ LÝ ASYNC:
   - Luôn sử dụng try-catch cho các async operations
   - Kiểm tra mounted trước khi setState trong async

2. QUẢN LÝ STATE:
   - Sử dụng setState để cập nhật UI
   - Kiểm tra _isGenerating để tránh multiple requests

3. ERROR HANDLING:
   - Hiển thị thông báo lỗi rõ ràng cho người dùng
   - Log lỗi chi tiết cho developer

4. PERFORMANCE:

   - Tránh rebuild không cần thiết
   - Cache API responses nếu có thể

5. SECURITY:
   - Không commit API keys vào git
   - Sử dụng secrets.dart hoặc environment variables
   - Validate user input trước khi gửi API
